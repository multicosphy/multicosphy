## 命名规则

- 课程项目：`Course-*`
- 研究项目：`Research-*`
- 论文探索：`Paper-*`
- 独立工具或实验：`Project-*`
- 阅读与笔记：`Reading-*`、`Notes-*`
- 个人兴趣：`Interest-*`

仓库名建议使用英文，README 标题和说明可以使用中文。

中文或中英混合仓库名在 URL、命令行、自动化脚本和跨平台路径处理上更容易出问题。
